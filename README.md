# Hospital Emergency Room Dashboard

This repository contains a **Power BI dashboard** developed to analyze hospital emergency room operations and patient flow. The dashboard provides insights into patient visits, wait times, admissions, and patient demographics to support healthcare decision-making.

## Dashboard Features

* Total number of patients visiting the emergency room
* Average patient wait time
* Patient satisfaction score
* Admission vs non-admission analysis
* Patient distribution by age group
* Patient distribution by gender and race
* Number of patients by day and hour
* Department referral analysis

## Tools Used

* Power BI
* Power Query
* DAX (Data Analysis Expressions)
* Excel / CSV dataset

## Files

* Hospital_ER_Dashboard.pbix
* dashboard_preview.png

## Purpose

The goal of this dashboard is to help healthcare administrators monitor emergency room performance, understand patient trends, and improve hospital service efficiency.
